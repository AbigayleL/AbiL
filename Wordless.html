Summary: Wordless

For this project, we were to create an interface that only uses sounds but not in the form of words or language as input. With the thought of the previous project, I wanted continue to explore design and animation in photoshop and use sounds and pictures. For this idea, the user will use the mic to hum, sing or play any single note on an instrument and the note will be shown both as a key and a note on the piano. If preferred, the user will be able to also hear feedback in the form of a piano key.

How to use

Note: Android Chrome use, please use headsets with mic

1: Press the use demo then the live input audio before both can work

2: Click the use live input and accept the use of mic

3: Hum, Sing, or play any single key on an instrument

4: Click for audio playback to hear notes (Warning: only use for single key instruments, keep volume low)
